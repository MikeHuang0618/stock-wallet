# UX 驗證截圖

本目錄存放各 Wave 的人工驗證截圖(`wave-N-*.png`)。

## 說明
Wave 6-8 的 UX 修繕在開發時以「靜態網頁伺服器 + 真實 HTML/CSS/JS + 注入 mock STATE」
於瀏覽器驗證(排版/顏色/動畫/行為),並以 pytest 驗證後端純邏輯(6.8 QuoteRefresher、
7.1 get_signals)。以下項目已於瀏覽器實測通過:

- 6.1 詳細頁 header 缺報價顯示「—」不殘留
- 6.3 現金餘額卡(現金 + 總市值 = 錢包價值)
- 6.4 錢包卡片 1240/990 兩寬度皆無孤兒卡(3 hero / 4 各幣別 / 6 各項數值)
- 6.5 漲跌顏色慣例切換(us 綠漲紅跌 ↔ tw 紅漲綠跌)
- 6.6 spark 顏色跟隨當日方向 + 30d 標示
- 6.2 sticky 收合(捲動 >200 收合成緊湊列、含現金,向上捲回展開)
- 6.9 卡片進場動畫只播一次(重繪加 .no-anim)

## 待補:真實桌面 App 截圖
`python app.py`(pywebview 桌面視窗)才能呈現原生視窗與橋接行為。請在真實 App
依各 commit message 的驗收清單操作後,將截圖存為 `wave-N-<項目>.png` 置於本目錄。
